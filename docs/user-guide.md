# User Guide

## Getting Started

### 1. Installation
1. Open Telegram and search for `@SimplRefQBot`
2. Click "Start" to begin
3. Follow the onboarding process

### 2. Initial Setup
1. Connect your wallet (TON/ETH/BNB)
2. Complete KYC verification
3. Set up 2FA for security
