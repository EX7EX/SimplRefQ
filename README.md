## Quick Start

1. Clone the repository:
```bash
git clone https://github.com/yourusername/SimplRefQ.git
cd SimplRefQ
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Set up environment variables:
```bash
cp .env.example .env
# Edit .env with your configuration
```

4. Run the bot:
```bash
python SimplRefQ.py
```
